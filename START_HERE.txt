========================================
  HEAD MOUSE + VOICE CONTROLLER
  QUICK START GUIDE
========================================

Hi! Welcome to the Head Mouse + Voice Controller.

This tool lets you control your computer using:
 - HEAD MOVEMENTS to move the cursor
 - EYE BLINKS for mouse clicks
 - VOICE COMMANDS for typing and apps

========================================
  WHAT YOU NEED
========================================

1. Windows 10/11 computer
2. Webcam (built-in or USB)
3. Microphone (for voice commands)
4. Python 3.8 or higher installed
5. Good lighting on your face

========================================
  INSTALLATION (One-Time Setup)
========================================

OPTION A - Easy Way (Windows):
1. Double-click "install_dependencies.bat"
2. Wait for installation to complete
3. Done!

OPTION B - Manual Way:
1. Open Command Prompt
2. Navigate to this folder:
   cd C:\path\to\eye_mouse_project
3. Run: pip install -r requirements.txt
4. Wait for completion

========================================
  HOW TO RUN
========================================

OPTION A - Easy Way (Windows):
1. Double-click "run_eye_mouse.bat"
2. Application opens!

OPTION B - Manual Way:
1. Open Command Prompt
2. Navigate to this folder
3. Run: python main.py

========================================
  FIRST TIME USE
========================================

1. CALIBRATE (Very Important!)
   - Click "Calibrate Gaze" button
   - Move your HEAD to position nose at each target
   - Blink both eyes when nose is at target
   - Complete all 5 points

2. START TRACKING
   - Click "Start Tracking" button
   - Move your HEAD to move cursor!
   - 2 blinks = Right click
   - 3 blinks = Left click
   - 4 blinks = Drag/Drop (grab and release)
   - 5 blinks = Middle click

3. USE VOICE ASSISTANT
   - Click "Enable Voice Assistant"
   - Click "üé§ Listen" button
   - Speak your command
   - Examples: "type hello", "open Chrome"

========================================
  CONTROLS (HEAD + VOICE!)
========================================

HEAD TRACKING:
Mouse Movement: Move your head left/right/up/down
Right Click:    2 blinks (both eyes, quick)
Left Click:     3 blinks (both eyes, quick)
Drag & Drop:    4 blinks to grab, move head, 4 blinks to drop
Middle Click:   5 blinks (open in new tab, etc.)

VOICE ASSISTANT:
Enable Voice:   Click "Enable Voice Assistant"
Listen:         Click "üé§ Listen" button
Commands:       
  - "type Hello World" - Type text hands-free
  - "open Chrome" - Launch Chrome browser
  - "open Notepad" - Launch Notepad
  - "open Calculator" - Launch Calculator
  - "copy" / "paste" - Keyboard shortcuts
  - "scroll down" / "scroll up" - Scroll pages
  - "search for Python" - Web search
  - "enter" / "backspace" / "delete" - Special keys
  - See FEATURES.md for all commands

Controls:
Pause:          Click "Pause" button
Exit:           Click "Exit" button or press Q

========================================
  TROUBLESHOOTING
========================================

Problem: "Python not recognized"
Solution: Install Python and check "Add to PATH"

Problem: "Camera not found"
Solution: Close other apps using webcam (Zoom, Skype)

Problem: Dependencies install fails
Solution: Update pip first:
          python -m pip install --upgrade pip

Problem: Cursor doesn't move
Solution: Recalibrate and ensure good lighting

Problem: Blinks don't register
Solution: Perform clear, full blinks (close eyes completely)
